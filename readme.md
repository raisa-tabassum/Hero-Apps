App Name: HERO.IO

Description:
HERO.IO is a web application designed to showcase a collection of productive apps that enhance digital experiences. Users can explore trending apps, search and filter apps, view detailed information, install or uninstall apps, and manage their installed applications. The app also features interactive charts for app ratings and provides a smooth, responsive experience across all devices.

Key Features:
* Browse trending apps with images, ratings, and download counts
* Live search and filter functionality
* Detailed app pages with description and ratings chart
* Install/Uninstall apps with LocalStorage support
* Sort apps by download count
* Responsive design for mobile, tablet, and desktop
* Custom error page for invalid routes


Technologies Used:
React – For building user interfaces  
React Router DOM – Handling routing and navigation  
Tailwind CSS – Utility-first CSS framework for styling  
DaisyUI – Component library built on Tailwind CSS  
Recharts – Charts for visualizing app ratings  
React Icons – Icons for buttons and UI elements  
Axios – Fetching data from JSON files  
React Toastify – Toast notifications for actions like Install  
LocalStorage – Client-side storage for installed apps
